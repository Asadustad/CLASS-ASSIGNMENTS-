For Each VS. Map

For Each

  • Purpose: Used to perform an action on each element of an array.
  • Return: It does not return anything.
  • Use case: When you want to do something with each item, but you dont need a new array.
const numbers = [1, 2, 3, 4, 5];
numbers.forEach(number => console.log(number * 2));
// Output: 2
4
6
8
10

Map

  • Purpose: Used to transform each element and create a new array.
  • Return: Returns a new array with the results of the transformation.
  • Use case: When you want to change each item and get a new array in return.
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map(number => number * 2);
console.log(doubled);
// Output: [2, 4, 6, 8, 10]
