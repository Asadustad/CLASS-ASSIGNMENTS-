.Some() .Filter() .Find() .Reduce()

SOME

  • 1. Check karta hai ke array ka koi element condition ko satisfy karta hai ya nah.
  • 2. Jald result deta hai — jaise hi pehla matching element milta hai, search ruk jata hai.
  • 3. Return karta hai true ya false, yani condition puri hui ya nahi.
const numbers = [1, 2, 3, 4];
const hasEven = numbers.some(num => num % 2 ===0);
// output :true

FILTER

  • 1. Sare matching elements ko nikalta hai aur unka naya array banata hai.
  • 2. Original array ko change nahi karta — naya array banata hai.
  • 3. Har element ko check karta hai aur sirf jo condition puri karte hain, unko return karta hai.
const numbers = [1, 2, 3, 4];
const evens = numbers.filter(num => num % 2 ===0);
//output
[2,4]

FIND

  • 1. Array me se pehla element return karta hai jo condition puri karta hai.
  • 2. Sirf ek element return hota hai — pehla match milne ke baad ruk jata hai.
  • 3. Agar koi bhi match nahi mila to undefined return karta hai.
const numbers = [1, 2, 3, 4];
const firstEven = numbers.find(num => num %2===0);
//output
2
2 br

REDUCE

  • 1. Array ke tamam elements ko mila kar ek hi value me convert karta hai (jaise sum, average, etc).
  • 2. Ek accumulator use karta hai — pehle se value store karke agle element ke sath process karta hai.
  • 3. Aap initial value bhi de sakte hain (jaise 0 ya {}), aur ye step-by-step array process karta hai.
const numbers = [1, 2, 3, 4];
const sum = numbers.reduce((acc, num) => acc +num,0);
//output
10
